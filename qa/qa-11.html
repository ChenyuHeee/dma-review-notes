什么是 Lattice（格）？

2026-06-27
Definition

A poset $(A, \preceq)$ is a lattice if every pair of elements has both a least upper bound (LUB) and a greatest lower bound (GLB).

翻译：任取两个元素，它们的最小上界和最大下界都存在，就是 lattice。

理解 lattice 只需要看一个例子

$A = \{1, 2, 3, 4, 6, 12\}$，偏序为整除 $a \mid b$。

任取两个元素，算 LUB 和 GLB：

$a, b$LUB = $\operatorname{lcm}(a,b)$在 $A$ 中？GLB = $\gcd(a,b)$在 $A$ 中？
$2, 6$$\operatorname{lcm}=6$✅$\gcd=2$✅
$3, 4$$\operatorname{lcm}=12$✅$\gcd=1$✅
$4, 6$$\operatorname{lcm}=12$✅$\gcd=2$✅
$2, 3$$\operatorname{lcm}=6$✅$\gcd=1$✅
$1, 12$$\operatorname{lcm}=12$✅$\gcd=1$✅

所有对的 LUB 和 GLB 都在 $A$ 中 → $(A, \mid)$ 是 lattice。

反例：什么不是 lattice

$B = \{2, 3, 4, 6\}$，整除偏序。

$a, b$LUB = $\operatorname{lcm}(a,b)$在 $B$ 中？
$3, 4$$\operatorname{lcm}=12$❌ $12 \notin B$

3 和 4 的最小上界 12 不在 $B$ 中 → 不是 lattice。

只要有一个元素对的 LUB 或 GLB 不在集合中，就不是 lattice。

判断流程

1. 在给定偏序下，LUB 和 GLB 的计算公式是什么？（整除偏序→lcm/gcd，子集偏序→并/交）

2. 任取一对元素，算出 LUB 和 GLB。

3. 检查是否都在集合中。全都满足 → lattice；有一个不在 → 不是。
