什么是 Equivalence Relation（等价关系）？

2026-06-27 · 离散数学 · 考前复习

一句话

等价关系 = 同时满足 Reflexive + Symmetric + Transitive 三种性质的关系。

你可以把它理解为一种分类规则——把集合中的元素分成若干组，同组的元素彼此"等价"。

三个条件

Reflexive（自反）
每个元素和自己等价
$(a,a) \in R$ 对所有 $a$
Symmetric（对称）
等价是双向的
若 $aRb$ 则 $bRa$
Transitive（传递）
等价可以传递
若 $aRb$ 且 $bRc$，则 $aRc$

经典例子

例 1：模 $m$ 同余

$a \equiv b \pmod{m}$ 是等价关系：

  • 自反：$a \equiv a \pmod{m}$ ✓
  • 对称：若 $a \equiv b$，则 $b \equiv a$ ✓
  • 传递：若 $a \equiv b$ 且 $b \equiv c$，则 $a \equiv c$ ✓

等价类：$[0]_3 = \{\dots,-3,0,3,6,\dots\}$（除以 3 余 0 的所有数）

例 2：等长字符串

字符串 $s$ 和 $t$ 等价 $\iff$ $|s| = |t|$（长度相同）

等价类：所有长度为 0 的串、所有长度为 1 的串、……

反例：整除关系 $a \mid b$

$3 \mid 6$ 但 $6 \nmid 3$ → 不对称，不是等价关系。

核心定理：等价类 = 划分

一个等价关系会把集合切成若干块（等价类），这些块：

  • 两两不相交
  • 并起来等于全集

这就叫 Partition（划分）。反之，给定一个划分，也可以定义一个等价关系。

等价关系 ⟷ 划分（一一对应）

这是 Ch9.5 最重要的定理，考试必考。
