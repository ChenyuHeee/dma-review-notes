Tree Traversal（树的遍历）

2026-06-27

同一棵树，三种走法

A / \ B C / \ \ D E F
遍历方式英文规则结果
前序Preorder根 → 左 → 右A, B, D, E, C, F
中序Inorder左 → 根 → 右D, B, E, A, C, F
后序Postorder左 → 右 → 根D, E, B, F, C, A

一句话记住

三种遍历的区别就是根节点什么时候访问：

Preorder：先访问根，再递归左右。（Pre = 根在最前）

Inorder：先递归左，再访问根，再递归右。（In = 根在中间）

Postorder：先递归左右，最后访问根。（Post = 根在最后）

表达式树（考试常考）

运算符在内部节点，操作数在叶子：

+ / \ * - / \ \ 2 3 4
遍历结果对应表达式
Preorder+ * 2 3 - 4Prefix（波兰表示法）
Inorder2 * 3 + - 4Infix（中缀，需加括号）
Postorder2 3 * 4 - +Postfix（逆波兰表示法）
考点

给一个表达式树，写出三种遍历结果。或反过来：给 prefix/postfix 表达式，画出表达式树。
