Partial Order（偏序）到底是什么？

2026-06-27

先忘掉数学定义，想一个现实例子

你手机里有几个 App。有些 App 之间可以比较"大小"（比如按安装时间），但有些 App 无法比较——比如微信和支付宝，谁的版本号更大？没法比。

Partial Order 就是"部分可比、部分不可比"的排序关系。

正式定义

集合 $A$ 上的关系 $R$ 是 Partial Order（偏序），必须满足三个条件：

条件含义例子: $\le$ 在整数上
① Reflexive$aRa$$3 \le 3$ ✓
② Antisymmetric若 $aRb$ 且 $bRa$，则 $a=b$$3 \le 5$ 且 $5 \le 3$ 不可能同时成立 ✓
③ Transitive若 $aRb$ 且 $bRc$，则 $aRc$$3 \le 5$ 且 $5 \le 7$ → $3 \le 7$ ✓

最关键的一点：Antisymmetric ≠ "不对称"

Symmetric（对称）

$aRb \Rightarrow bRa$

你爱我 = 我爱你
等价关系用这个

Antisymmetric（反对称）

$aRb$ 且 $bRa \Rightarrow a = b$

双向箭头只能出现在自己和自己的自环上
偏序用这个

记忆：偏序 = "有方向的层级关系"，不允许两个不同的元素互相指向对方。所以它叫 Anti-symmetric。

经典例子对比

例 1：$(\mathbb{Z}, \le)$ — 全序（链）

任意两个数都可以比较：$3 \le 5$ 或 $5 \le 3$ 必有一个成立。

这种情况叫 Total Order（全序），是偏序的一种特殊情况。

例 2：$(\mathbb{Z}^+, \mid)$ — 整除偏序（考试最爱）

$a R b \iff a \mid b$（$a$ 整除 $b$）

$A = \{1, 2, 3, 4, 6, 12\}$：

  • $2 \mid 4$ ✓（2 在 4 "下面"）
  • $2 \mid 6$ ✓
  • $3$ 和 $4$ 之间？$3 \nmid 4$，$4 \nmid 3$ → 不可比较！

这种"有些能比有些不能比"的关系就是偏序。

这就解释了为什么叫 Partial（部分的，不完整的）——不是所有元素都能比较。

例 3：$(\mathcal{P}(\{a,b\}), \subseteq)$ — 子集偏序

$\emptyset \subseteq \{a\} \subseteq \{a,b\}$，但 $\{a\}$ 和 $\{b\}$ 互不包含 → 不可比较。
