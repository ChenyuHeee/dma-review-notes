NAND / NOR / XOR / CNF / DNF

2026-06-27

特殊逻辑运算符

符号名称定义真值
$p \mid q$NAND（与非）$\neg(p \land q)$仅有 (T,T) → F，其余 T
$p \downarrow q$NOR（或非）$\neg(p \lor q)$仅有 (F,F) → T，其余 F
$p \oplus q$XOR（异或）$(p \land \neg q) \lor (\neg p \land q)$不同 → T，相同 → F

功能完备性：$\{\mid\}$ 和 $\{\downarrow\}$ 各自单独就能表达所有逻辑运算。

  • $\neg p \equiv p \mid p$（NAND 自己和自己 = NOT）
  • $p \land q \equiv \neg(p \mid q) \equiv (p \mid q) \mid (p \mid q)$
  • $p \lor q \equiv (p \mid p) \mid (q \mid q)$

CNF 和 DNF（考试每年必考一道大题）

DNF（析取范式）：若干 minterm（文字的合取）的析取 $\equiv$ 积之和

CNF（合取范式）：若干 maxterm（文字的析取）的合取 $\equiv$ 和之积

Minterm：每个变量恰好出现一次，全部用 $\land$ 连接。
Maxterm：每个变量恰好出现一次，全部用 $\lor$ 连接。

怎么做：真值表法（最可靠）

求 $f = p \oplus q$ 的 Full DNF：

$p$$q$$p \oplus q$minterm
FFF—
FTT$\neg p \land q$
TFT$p \land \neg q$
TTF—

Full DNF = 所有为 T 的行对应的 minterm 取 $\lor$：

$$f = (\neg p \land q) \lor (p \land \neg q)$$

Full CNF = 所有为 F 的行对应的 maxterm 取 $\land$：

$$f = (p \lor q) \land (\neg p \lor \neg q)$$
背下来

DNF = 找真值表中为 T 的行 → 每行写 minterm（T 用原变量，F 用否定）→ 全部 $\lor$ 起来。

CNF = 找真值表中为 F 的行 → 每行写 maxterm（F 用原变量，T 用否定）→ 全部 $\land$ 起来。
