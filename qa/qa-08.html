Comparability & Lexicographic Order

2026-06-27

Comparability（可比性）

Definition

在偏序 $(A, \preceq)$ 中，两个元素 $a, b \in A$ 是 comparable 如果：

$a \preceq b$ 或 $b \preceq a$

否则称为 incomparable（不可比）。

例子：$A = \{1,2,3,4,6,12\}$，偏序为整除 $a \mid b$。

元素对关系结果
2, 4$2 \mid 4$comparable ✅
3, 6$3 \mid 6$comparable ✅
3, 4$3 \nmid 4$ 且 $4 \nmid 3$incomparable ❌
2, 12$2 \mid 12$comparable ✅

Total Order（全序）vs Partial Order（偏序）

如果偏序中任意两个元素都可比，就叫 total order（全序，也叫 linear order / 线序）。

如果存在不可比的元素对，就只是 partial order（偏序）。

例子任意两个可比？类型
$(\mathbb{Z}, \le)$$3 \le 5$ 或 $5 \le 3$ 总有一个成立是Total Order
$(\{1,2,3,4,6,12\}, \mid)$$3$ 和 $4$ 无法比较否Partial Order

Lexicographic Order（字典序）

Definition

在两个偏序 $(A_1, \preceq_1)$ 和 $(A_2, \preceq_2)$ 的笛卡尔积 $A_1 \times A_2$ 上定义字典序 $\preceq$：

$(a_1, a_2) \prec (b_1, b_2) \iff (a_1 \prec_1 b_1) \;\text{或}\; (a_1 = b_1 \;\text{且}\; a_2 \prec_2 b_2)$

简单说：先比第一个分量，第一个相同再比第二个。

例子：字符串的字典序。

$A = \{a,b,c,\dots,z\}$，偏序为字母表顺序。扩展到 $A \times A \times \dots$ 上：

"bat" $\prec$ "cat"（第一个字母 b < c，直接决定）

"car" $\prec$ "cat"（前两个字母 ca = ca，第三个 r < t 决定）

考试形式：给你两个元组，判断谁在字典序中更小。

$(2, 5, 3) \prec (2, 5, 7)$：$2=2$, $5=5$, $3<7$ → $(2,5,3) \prec (2,5,7)$。

$(2, 8, 1) \prec (3, 1, 9)$：$2<3$ → 直接结束，$(2,8,1) \prec (3,1,9)$。
