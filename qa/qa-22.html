Part 1: Injective / Surjective / Bijective

Injective（单射）

不同输入 → 不同输出

$f(x)=f(y) \Rightarrow x=y$

Surjective（满射）

陪域中每个值都被打到

$\forall y \in B, \exists x \in A: f(x)=y$

Bijective（双射）

单射 + 满射

一一对应

$|A|$ 是什么意思

$|A|$ = 集合 $A$ 的 cardinality（基数） = $A$ 中元素的个数。有限集就是一个数字。无限集用 $\aleph_0$（可数无穷）等表示。

$|A \times B| = |A| \cdot |B|$：笛卡尔积的基数 = 两个基数的乘积。

Part 2: Big-O / Big-Ω / Big-Θ

记号含义定义
$f = O(g)$$f$ 增长 不快于 $g$$\exists C, n_0: f(n) \le C \cdot g(n)$ for $n \ge n_0$
$f = \Omega(g)$$f$ 增长 不慢于 $g$$\exists C, n_0: f(n) \ge C \cdot g(n)$ for $n \ge n_0$
$f = \Theta(g)$$f$ 和 $g$ 同阶$f = O(g)$ 且 $f = \Omega(g)$
$f = o(g)$$f$ 增长 严格慢于 $g$$\lim_{n\to\infty} f(n)/g(n) = 0$

记忆：O = 上界（≤），Ω = 下界（≥），Θ = 紧界（=），o = 严格上界（<）。

例子：$f(n) = 3n^2 + 2n + 1$

$f = O(n^2)$ ✓（$3n^2+2n+1 \le 6n^2$ for large $n$）

$f = \Omega(n^2)$ ✓（$3n^2+2n+1 \ge 3n^2$）

$f = \Theta(n^2)$ ✓（上下界都是 $n^2$）

$f = o(n^3)$ ✓（$\lim f/n^3 = 0$）

常见函数增长率（从慢到快）

常数 < $\log n$ < $(\log n)^2$ < $\sqrt{n}$ < $n$ < $n\log n$ < $n^2$ < $n^3$ < $2^n$ < $n!$
