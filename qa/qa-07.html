Partial Order（偏序）定义

2026-06-27
Definition

A relation $R$ on a set $A$ is a partial order if it is:

1. Reflexive: $(a,a) \in R$ for all $a \in A$

2. Antisymmetric: if $(a,b) \in R$ and $(b,a) \in R$, then $a = b$

3. Transitive: if $(a,b) \in R$ and $(b,c) \in R$, then $(a,c) \in R$

集合 $A$ 连同其上的偏序 $R$ 称为 poset（partially ordered set），记作 $(A, R)$。

三个条件逐一解释

Reflexive（自反）——每个元素必须和自己有关系。这是形式要求，几乎总是满足的。

Antisymmetric（反对称）——这是偏序的核心特征。看逻辑结构：

IF (a,b) ∈ R AND (b,a) ∈ R THEN a = b

"同一个箭头不能双向存在，除非是自环"。等价关系要求 Symmetric（有来有回），偏序要求 Antisymmetric（有来无回）。只有这个条件不同。

Transitive（传递）——如果 a 指向 b 且 b 指向 c，则 a 必须直接指向 c。和等价关系的传递完全一样。

两个标准例子

例 1：$A = \mathbb{Z}$，$R$ 为 $\le$（小于等于）

自反：$a \le a$ ✓ | 反对称：$a \le b$ 且 $b \le a \Rightarrow a = b$ ✓ | 传递：$a \le b$ 且 $b \le c \Rightarrow a \le c$ ✓

例 2：$A = \{1,2,3,4,6,12\}$，$R$ 为整除 $a \mid b$

自反：$a \mid a$ ✓ | 反对称：$a \mid b$ 且 $b \mid a \Rightarrow a=b$ ✓ | 传递：$a \mid b$ 且 $b \mid c \Rightarrow a \mid c$ ✓

这个例子中 $3$ 和 $4$ 之间没有任何整除关系。这就是 Partial 的含义——不是任意两个元素都可比。
