# Chapter 2.3: Functions (函数)

---

## 1. Definition of a Function (函数的定义)

**Function (函数)**: Let $A$ and $B$ be nonempty sets. A **function** $f$ from $A$ to $B$, denoted $f: A \to B$, is an assignment of each element of $A$ to **exactly one** element of $B$.

$$f(a) = b \text{ means } b \text{ is the unique element assigned to } a$$

Formally:
$$\forall a (a \in A \to \exists! b (b \in B \land f(a) = b))$$

- Functions are sometimes called **mappings (映射)** or **transformations (变换)**.
- A function $f: A \to B$ can also be defined as a subset of $A \times B$ (a relation), where no two ordered pairs have the same first element.

---

## 2. Domain, Codomain, Image, Preimage (定义域、陪域、像、原像)

For a function $f: A \to B$:

| Term | Notation | Meaning |
|------|----------|---------|
| **Domain (定义域)** | $A$ | The set of all inputs |
| **Codomain (陪域)** | $B$ | The set of all possible outputs |
| **Image (像)** of $a$ | $f(a)=b$ | The output corresponding to input $a$ |
| **Preimage (原像)** of $b$ | $a$ | An input whose output is $b$ |
| **Range (值域)** | $f(A)$ | The set of all actual outputs: $\{f(a) \mid a \in A\}$ |

> **Important**: Codomain 是"可能输出的集合"，Range 是"实际输出的集合"。Range $\subseteq$ Codomain.

### Example
```
A = {a,b,c,d}    B = {x,y,z}
f(a) = x, f(b) = y, f(c) = z, f(d) = z
```
- Domain = $A$, Codomain = $B$
- Image of $d$ = $z$, Preimage(s) of $z$ = $\{c,d\}$
- Range $f(A) = \{x, y, z\}$ (actually all of $B$ here)

### Image of a Subset (子集的像)
If $S \subseteq A$, then:
$$f(S) = \{f(s) \mid s \in S\}$$

In the example above: $f(\{a,b,c\}) = \{x, y, z\}$, $f(\{c,d\}) = \{z\}$.

## 3. Types of Functions (函数的类型)

### Injective (One-to-One) — 单射
**Definition**: A function $f$ is **one-to-one** or **injective (单射)** iff $f(a) = f(b)$ implies $a = b$ for all $a, b$ in the domain.

$$f \text{ is injective} \iff \forall a \forall b (f(a) = f(b) \to a = b)$$

等价说法：不同的输入映射到不同的输出。

**How to prove injectivity**: Assume $f(a) = f(b)$ and prove $a = b$.

### Surjective (Onto) — 满射
**Definition**: A function $f: A \to B$ is **onto** or **surjective (满射)** iff for every element $b \in B$, there is an element $a \in A$ with $f(a) = b$.

$$f \text{ is surjective} \iff \forall b \in B, \exists a \in A \text{ such that } f(a) = b$$

等价说法：值域等于陪域，即 $f(A) = B$。

**How to prove surjectivity**: For an arbitrary $b \in B$, find (construct) an $a \in A$ such that $f(a) = b$.

### Bijective (One-to-One Correspondence) — 双射
**Definition**: A function $f$ is a **one-to-one correspondence** or a **bijection (双射)** if it is **both** injective and surjective.

---

## 4. Examples and Counterexamples

**Example 1**: $f: \{a,b,c,d\} \to \{1,2,3\}$ with $f(a)=3, f(b)=2, f(c)=1, f(d)=3$
- Is $f$ onto? **Yes** — all three elements of codomain are images.
- If codomain were $\{1,2,3,4\}$, $f$ would **not** be onto.

**Example 2**: $f(x) = x^2$ from $\mathbb{Z}$ to $\mathbb{Z}$
- Not onto: no integer $x$ with $x^2 = -1$.
- Not one-to-one: $f(2) = f(-2) = 4$ but $2 \neq -2$.

---

## 5. Inverse Functions (反函数)

**Definition**: Let $f$ be a **bijection** from $A$ to $B$. Then the **inverse** of $f$, denoted $f^{-1}$, is the function from $B$ to $A$ defined by:
$$f^{-1}(b) = a \iff f(a) = b$$

> **No inverse exists unless $f$ is a bijection.** Why?
> - If $f$ is not injective, then $f^{-1}$ would have to map one output to multiple inputs — not a function.
> - If $f$ is not surjective, then some elements of $B$ have no preimage — $f^{-1}$ would be undefined for them.

**Examples:**

1. $f: \{a,b,c\} \to \{1,2,3\}$, $f(a)=2, f(b)=3, f(c)=1$
   - $f$ is bijective. $f^{-1}(1)=c, f^{-1}(2)=a, f^{-1}(3)=b$

2. $f: \mathbb{Z} \to \mathbb{Z}$, $f(x) = x + 1$
   - $f$ is bijective. $f^{-1}(y) = y - 1$

3. $f: \mathbb{R} \to \mathbb{R}$, $f(x) = x^2$
   - $f$ is **not** invertible (not one-to-one, e.g., $f(2)=f(-2)=4$).

---

## 6. Function Composition (函数组合)

**Definition**: Let $f: B \to C$ and $g: A \to B$. The **composition** of $f$ with $g$, denoted $f \circ g$, is the function from $A$ to $C$ defined by:
$$(f \circ g)(x) = f(g(x))$$

> **注意顺序**: $f \circ g$ 表示先应用 $g$，再应用 $f$。

**Examples:**

1. $f(x) = x + 1$, $g(x) = x^2$
   - $(f \circ g)(x) = f(g(x)) = f(x^2) = x^2 + 1$
   - $(g \circ f)(x) = g(f(x)) = g(x+1) = (x+1)^2 = x^2 + 2x + 1$

2. $g: \{a,b,c\} \to \{a,b,c\}$, $g(a)=b, g(b)=c, g(c)=a$
   $f: \{a,b,c\} \to \{1,2,3\}$, $f(a)=3, f(b)=2, f(c)=1$
   - $f \circ g$: $(f \circ g)(a) = f(g(a)) = f(b) = 2$, $(f \circ g)(b) = f(g(b)) = f(c) = 1$, $(f \circ g)(c) = f(g(c)) = f(a) = 3$
   - $g \circ f$ is **not defined** because range of $f$ ($\{1,2,3\}$) is not a subset of domain of $g$ ($\{a,b,c\}$).

3. $f(x) = 2x + 3$, $g(x) = 3x + 2$ (both $\mathbb{Z} \to \mathbb{Z}$)
   - $(f \circ g)(x) = f(g(x)) = f(3x+2) = 2(3x+2) + 3 = 6x + 7$
   - $(g \circ f)(x) = g(f(x)) = g(2x+3) = 3(2x+3) + 2 = 6x + 11$

---

## 7. Graphs of Functions (函数图像)

The **graph** of a function $f: A \to B$ is the set of ordered pairs:
$$\{(a, b) \mid a \in A \text{ and } f(a) = b\}$$

Examples:
- $f(x) = x^2$ from $\mathbb{Z}$ to $\mathbb{Z}$: graph is $\{(n, n^2) \mid n \in \mathbb{Z}\}$
- $f(n) = 2n + 1$ from $\mathbb{Z}$ to $\mathbb{Z}$: graph is $\{(n, 2n+1) \mid n \in \mathbb{Z}\}$

## 8. Floor and Ceiling Functions (取整函数)

### Floor Function $\lfloor x \rfloor$
The **floor** of $x$ is the **largest integer** less than or equal to $x$.
$$\lfloor x \rfloor = \max\{n \in \mathbb{Z} \mid n \le x\}$$

"向下取整"：不超过 $x$ 的最大整数。

### Ceiling Function $\lceil x \rceil$
The **ceiling** of $x$ is the **smallest integer** greater than or equal to $x$.
$$\lceil x \rceil = \min\{n \in \mathbb{Z} \mid n \ge x\}$$

"向上取整"：不小于 $x$ 的最小整数。

| $x$ | $\lfloor x \rfloor$ | $\lceil x \rceil$ |
|:---:|:---:|:---:|
| 2.3 | 2 | 3 |
| -2.3 | -3 | -2 |
| 5 | 5 | 5 |
| 0.5 | 0 | 1 |
| -0.5 | -1 | 0 |

### Proving Properties of Floor Functions

**Example**: Prove that for any real number $x$, $\lfloor 2x \rfloor = \lfloor x \rfloor + \lfloor x + 1/2 \rfloor$.

**Solution**: Let $x = n + \varepsilon$, where $n$ is an integer and $0 \le \varepsilon < 1$.

**Case 1**: $\varepsilon < 1/2$
- $2x = 2n + 2\varepsilon$ and $\lfloor 2x \rfloor = 2n$ (since $0 \le 2\varepsilon < 1$)
- $\lfloor x + 1/2 \rfloor = \lfloor n + (1/2 + \varepsilon) \rfloor = n$ (since $0 \le 1/2 + \varepsilon < 1$)
- Hence $\lfloor 2x \rfloor = 2n$ and $\lfloor x \rfloor + \lfloor x + 1/2 \rfloor = n + n = 2n$.

**Case 2**: $\varepsilon \ge 1/2$
- $2x = 2n + 2\varepsilon = (2n + 1) + (2\varepsilon - 1)$ and $\lfloor 2x \rfloor = 2n + 1$ (since $0 \le 2\varepsilon - 1 < 1$)
- $\lfloor x + 1/2 \rfloor = \lfloor n + (1/2 + \varepsilon) \rfloor = \lfloor n + 1 + (\varepsilon - 1/2) \rfloor = n + 1$ (since $0 \le \varepsilon - 1/2 < 1$)
- Hence $\lfloor 2x \rfloor = 2n + 1$ and $\lfloor x \rfloor + \lfloor x + 1/2 \rfloor = n + (n + 1) = 2n + 1$.

Both cases satisfy the equation, so the statement is proved.

## 9. Factorial Function (阶乘)

**Definition**: $f: \mathbb{N} \to \mathbb{Z}^+$, denoted $f(n) = n!$, is the product of the first $n$ positive integers.
$$n! = 1 \cdot 2 \cdot 3 \cdots (n-1) \cdot n, \quad 0! = 1$$

Examples:
- $1! = 1$
- $2! = 1 \cdot 2 = 2$
- $6! = 720$
- $20! = 2,432,902,008,176,640,000$

**Stirling's Formula** (approximation for large $n$):
$$n! \sim \sqrt{2\pi n}\left(\frac{n}{e}\right)^n$$

---

## 10. Partial Functions (偏函数, optional)

**Definition**: A **partial function** $f$ from $A$ to $B$ assigns each element $a$ in a **subset** of $A$ (called the **domain of definition**) to a unique element $b$ in $B$.

- $f$ is **undefined** for elements in $A$ not in the domain of definition.
- When the domain of definition equals $A$, we say $f$ is a **total function (全函数)**.

Example: $f: \mathbb{Z} \to \mathbb{R}$ where $f(n) = \sqrt{n}$ is a partial function. Domain of definition is nonnegative integers. $f$ is undefined for negative integers.

### Exam Tips / 考点提示

1. **判断函数类型**: 单射 ($f(a)=f(b) \to a=b$)，满射 (range = codomain)，双射 (both)。
2. **反函数存在的充要条件**: 函数必须是**双射**。
3. **复合函数的顺序**: $(f \circ g)(x) = f(g(x))$，先 $g$ 后 $f$。$f \circ g$ 和 $g \circ f$ 一般不同。
4. **复合函数定义条件**: range of $g$ 必须是 domain of $f$ 的子集。
5. **Floor/Ceiling 证明技巧**: 将 $x$ 分解为整数部分 + 小数部分 $x = n + \varepsilon$，分情况讨论 $\varepsilon < 1/2$ 和 $\varepsilon \ge 1/2$。
6. **负数的取整**: $\lfloor -2.3 \rfloor = -3$ (小于 $-2.3$ 的最大整数是 $-3$)，$\lceil -2.3 \rceil = -2$。
